# AnimatedAvatars
[![Download][icon]][link] 

- Gives abality to have gif as an avatar  
- Everyone with this plugin can see your avatar  
- To use one, open settings > plugins > Animated Avatars > settings > and paste .gif url  
- PNG, JPG, GIF, ... are supported
